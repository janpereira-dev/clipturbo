# Pipeline Media

## Etapas

1. idea aprobada
2. hook aprobado
3. guion versionado
4. voz seleccionada
5. audio generado
6. assets normalizados
7. subtitulos generados
8. render 9:16
9. revision humana
10. exportacion o publicacion

## Principios

- etapas aislables
- trazabilidad por etapa
- reintentos sin rehacer todo
- FFmpeg como base de render
- contratos estables entre pasos
